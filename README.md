# Arduino-Touchscreen-Streamdeck
An interactive streamdeck with the Arduino UNO and 2.8" TFT touchscreen.
Note: This GitHub page is still WIP
## Materials used:
* Arduino UNO https://docs.arduino.cc/hardware/uno-rev3/
* Elegoo 2.8" TFT Touchscreen https://us.elegoo.com/products/elegoo-2-8-inches-tft-touch-screen
* 3D-Printed Case
## Get started:
* Download Elegoo TFT libraries from their website and install them. https://www.elegoo.com/blogs/arduino-projects/elegoo-2-8-inch-touch-screen-for-raspberry-pi-manual
* Build and flash the streamdeckArduinoCode to the board
* run either deck.py or deck.exe (in dist/)
* to automatically start with system bootup you can put the script into the windows startup apps folder
